# 推荐使用 pycharm 打开项目至 /backend，因为已经内置 django 程序无需再在环境中下载

# Django方面

1. 建表与运行

```
// 进入 backend 文件夹，依次执行：
pip install pandas
pip install openpyxl

python manage.py runserver 127.0.0.1:8000 // 启动程序

```


# Vue方面

1. 运行

```
// 进入 /frontend 文件路径
yarn install // 下载依赖

yarn serve // 运行程序

// 在浏览器输入相应端口
```
